# How to use Schtroumpf social

Schtroumpf social is a ruby Twitter clone made for a school project at Toulouse Ynov Campus

## Dependencies

- Ruby 3.0.3
- Rails 7.0.2

## Launching app

```ruby
bundle install
rails db:migrate
rails s
```

## Navigation

Go to [localhost:3000](http://localhost:3000) in your navigator

- Log-in / Sign-in
- User profile page
- New post
- Main page
